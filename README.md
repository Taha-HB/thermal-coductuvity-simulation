# Thermal Conductivity Simulation

## Description
This project simulates heat transfer through different materials using
the heat equation. The effect of thermal conductivity on temperature
distribution over time is analyzed.

- Python (NumPy, Matplotlib)
- MATLAB

## Materials Simulated

- Aluminum
- Steel
- Wood
